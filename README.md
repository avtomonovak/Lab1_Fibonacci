# Fibonacci Calculator - Лабораторная работа №1

## Описание
Программа для вычисления n-го числа Фибоначчи с проверкой входных данных.

## Вариант
Вариант 1: Последовательность Фибоначчи

## Требования
- Компилятор C++17 (g++ 7+ или clang 5+)
- make
- (для deb-пакета) dpkg

## Сборка
```bash
make                # сборка программы
make test           # запуск тестов
make debug          # отладочная сборка
make clean          # очистка
make run            # запуск программы
